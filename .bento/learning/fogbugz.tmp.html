FogBugz offers project management capabilities to help us manage tasks, organize sprints, set up a help desk, and more.

How to create Case in Fogbugz

  1. Click on New case.

  2. Enter the case Title.

  3. Select Documentation from Project dropdown list.

  4. Select the Area and Category that corresponds to a particular case.

  5. Assign the case to a person or a team by selecting the Assinged To dropdown list.

  6. Add users in Notify More Users about the status of your project

  7. Set Priorities based on the task requirement.

  8. Add Parent cases and Subcases if applicable.

  9. Add due date as required.

  10. Click the Open button to create a case.
